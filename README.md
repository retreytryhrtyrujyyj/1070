# 婴幼儿产品销售平台

## 项目简介
婴幼儿商品销售平台是基于Java Web技术，SSH框架、Myqsl数据库的专门销售婴幼儿产品的平台。功能上分为：商品列表、商品详情、购物车、订单、个人中心等模块。后台管理系统可以对商品新增、编辑、删除等功能。 此项目有完整的配套文档，适合作为毕业设计、大作业、课程设计等
